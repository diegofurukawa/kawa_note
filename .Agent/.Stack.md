# TECHNOLOGY STACK

## Frontend

### Core
- **Framework:** React 18.2.0
- **Build Tool:** Vite 6.1.0
- **Package Manager:** npm

### UI & Styling
- **Tailwind CSS:** 3.4.17
- **Icons:** Lucide React
- **Components:** Radix UI / Shadcn UI
- **Animations:** Framer Motion

### State & Data
- **Fetching:** @tanstack/react-query 5.84.1
- **Forms:** React Hook Form + Zod
- **Routing:** React Router DOM 6.26.0

### Utilities
- **Dates:** date-fns, moment
- **Helpers:** lodash, clsx, tailwind-merge
- **Confetti:** canvas-confetti

### External Integrations
- **Payments:** Stripe (@stripe/stripe-js)
- **Maps:** Leaflet (react-leaflet)
- **3D:** Three.js
- **PDF/Canvas:** jspdf, html2canvas

## Backend (Implemented)

### Core
- **Runtime:** Node.js 20 LTS (Alpine)
- **Framework:** Fastify 4
- **Database:** PostgreSQL 16 (Alpine)
- **ORM:** Prisma 5

### Libraries
- **Validation:** Zod
- **Auth:** JWT + bcrypt
- **Logging:** Pino
- **Testing:** Vitest + Supertest

### API Modules
- **Auth:** `/api/auth` - Register, Login, Me, Logout, Refresh
- **Notes:** `/api/notes` - CRUD + Search
- **Folders:** `/api/folders` - CRUD + Hierarchy
- **Relations:** `/api/relations` - CRUD + Graph
- **Apps Public:** `/api/apps/public` - Public settings

### Docker Architecture
- **Multi-stage Dockerfile:** deps → build → runtime
- **Services:** PostgreSQL + API + Migration (manual)
- **Network:** kawatech-network (10.10.0.0/16)
- **Static IPs:** API (10.10.0.31), DB (10.10.0.32)

## Infrastructure

### Production
- **Host:** Ubuntu Server LTS
- **Container Runtime:** Docker Standalone
- **Orchestration:** Portainer CE/BE
- **Network:** Docker Bridge (kawatech-network, 10.10.0.0/16)

### Deployment
- **Strategy:** Build on same host (no external registry)
- **Frontend:** Multi-stage Dockerfile (node:20-alpine → nginx:alpine)
- **Backend:** Multi-stage Dockerfile (node:20-alpine)
- **Config:** Environment variables via Portainer UI (no versioned .env)
