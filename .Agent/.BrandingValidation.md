## ðŸ”§ Common Issues & Fixes

### Issue: OG Image Not Displaying
**Cause:** Image URL incorrect or image too large  
**Fix:**
1. Verify image URL is correct
2. Check image size (<1MB)
3. Clear cache in debugger
4. Re-test after 24 hours

### Issue: Meta Description Truncated
**Cause:** Description too long (>160 chars)  
**Fix:**
1. Shorten description to 150-160 chars
2. Include primary keyword
3. Include CTA
4. Re-test

### Issue: Mobile Layout Broken
**Cause:** Viewport meta tag missing or incorrect  
**Fix:**
1. Verify viewport meta tag: `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
2. Test on actual mobile device
3. Check CSS media queries
4. Use Chrome DevTools mobile emulation

### Issue: Core Web Vitals Poor
**Cause:** Large images, slow JavaScript, layout shifts  
**Fix:**
1. Optimize images (use WebP, compress)
2. Defer non-critical JavaScript
3. Use CSS containment to prevent layout shifts
4. Lazy-load images below fold

### Issue: SSL Certificate Error
**Cause:** Certificate expired or misconfigured  
**Fix:**
1. Check certificate expiration date
2. Renew certificate if needed
3. Verify HTTPS redirect
4. Test with SSL Labs
