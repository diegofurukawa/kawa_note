# AGENT GOALS

## Objetivo Geral
Manter e evoluir o sistema **KawaMyCenter** de forma segura, performática e seguindo rigorosamente as convenções estabelecidas.

## Objetivos Específicos
1. **Qualidade de Código:** Garantir que 100% das alterações passem pelas regras de lint e revisão.
2. **Segurança:** Nunca expor chaves ou dados sensíveis.
3. **Documentação:** Manter a Stack, Convenções e Branding sempre atualizadas conforme o código evolui.
4. **Autonomia Controlada:** Agir proativamente como Auditor para propor melhorias, mas aguardar aprovação explícita para execução de planos complexos.
5. **Identidade de Marca:** Manter consistência de branding, SEO e posicionamento em todos os touchpoints.
