## Strategic Foundation

### Brand Identity
- **Name:** Kawa Note
- **Tagline:** "Suas notas. Suas conexões. Só suas." (Your notes. Your connections. Only yours.)
- **Category:** Privacy-First Knowledge Management Platform
- **Target Audience:** Privacy-conscious professionals, researchers, students, developers

### Market Position
- **Positioning:** Premium privacy-focused alternative to Notion/Obsidian
- **Differentiator:** E2E encryption + semantic note relationships + multi-tenant ready
- **Tone:** Technical, trustworthy, privacy-first, modern, clear
- **Emotional Drivers:** Control, clarity, security, connection

### Visual Language
- **Primary Color:** #0f766e (Teal 700) — Trust, security, tech
- **Accent Color:** #06b6d4 (Cyan 500) — Connection nodes, highlights
- **Typography:** Geist Sans / Inter (modern, geometric)
- **Aesthetic:** Minimalist, technical, accessible

## Logo System

### Primary Logo (logo.svg)
- **Concept:** Stylized "K" with connection nodes (knowledge graph)
- **Colors:** Teal (#0f766e) + Cyan (#06b6d4)
- **Scalability:** 32px minimum (web), 16px (favicon)
- **Usage:** Light backgrounds, web app header, landing page

### Dark Variant (logo-dark.svg)
- **Concept:** Same geometry, light colors
- **Colors:** Light cyan (#e0f2fe) + Cyan (#06b6d4)
- **Usage:** Dark backgrounds, dark mode UI

### Icon Version (icon.svg)
- **Concept:** Simplified "K" symbol
- **Size:** 64x64 (favicon-ready)
- **Usage:** Favicon, social profiles, app icon

### Geometry Rationale
- **K Symbol:** Represents the product name (Kawa Note)
- **Connection Nodes:** Represent knowledge graph (semantic connections)
- **Teal + Cyan:** Trust (teal) + Connection (cyan)
- **Geometric Construction:** Modern, scalable, professional

## SEO & Metadata System

### Core Meta Tags
```html
<title>Kawa Note - Encrypted Note-Taking with Knowledge Graph</title>
<meta name="description" content="End-to-end encrypted note-taking app with semantic connections...">
<meta name="robots" content="index, follow, max-image-preview:large">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta charset="UTF-8">
```

### Open Graph (Social Sharing)
```html
<meta property="og:title" content="Kawa Note - Your Notes. Your Connections. Only Yours.">
<meta property="og:description" content="End-to-end encrypted notes with semantic connections...">
<meta property="og:type" content="website">
<meta property="og:url" content="https://kawanote.app">
<meta property="og:image" content="https://kawanote.app/og-image.png">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta property="og:site_name" content="Kawa Note">
<meta property="og:locale" content="pt_BR">
```

### Twitter Card
```html
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:site" content="@kawanote">
<meta name="twitter:title" content="Kawa Note - Encrypted Note-Taking">
<meta name="twitter:description" content="End-to-end encrypted notes with semantic connections...">
<meta name="twitter:image" content="https://kawanote.app/og-image.png">
```

### JSON-LD Schemas
- **SoftwareApplication:** App metadata, ratings, features
- **Organization:** Company info, contact, social links
- **Website:** Search action, breadcrumbs
- **BreadcrumbList:** Navigation hierarchy

### SEO Keywords
- **Primary:** encrypted notes, privacy notes, knowledge graph, note-taking app
- **Secondary:** end-to-end encryption, secure notes, personal knowledge management
- **Long-tail:** encrypted notes for researchers, knowledge graph for students, etc.
