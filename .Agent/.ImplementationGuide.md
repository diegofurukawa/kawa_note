## üé® Phase 2: Assets (‚è≥ NEXT ‚Äî Week 2)

### Deliverables
- [ ] favicon.ico (16x16, 32x32, 64x64)
- [ ] og-image.png (1200x630)
- [ ] og-image-square.png (800x800)
- [ ] manifest.json (PWA metadata)
- [ ] robots.txt (SEO directives)

### Step-by-Step

#### 1. Generate Favicon
**Option A: Online Tool**
1. Go to https://favicon-generator.org/
2. Upload icon.svg
3. Generate favicon.ico
4. Download and save to `frontend/public/favicon.ico`

**Option B: ImageMagick (CLI)**
```bash
cd frontend/public
convert icon.svg -define icon:auto-resize=16,32,48,64 favicon.ico
```

**Option C: Manual (Figma)**
1. Open icon.svg in Figma
2. Export as PNG (16x16, 32x32, 64x64)
3. Use online tool to convert to ICO

#### 2. Design OG Images
**Option A: Figma (Recommended)**
1. Create new file in Figma
2. Create 1200x630 artboard
3. Add teal gradient background (#0f766e ‚Üí #0d9488)
4. Import logo.svg (centered, 200-300px)
5. Add text: "Your notes. Your connections. Only yours." (white, 48-60px)
6. Add cyan connection nodes (#06b6d4) as decoration
7. Export as PNG (og-image.png)
8. Create 800x800 variant (og-image-square.png)

**Option B: Canva**
1. Go to https://canva.com
2. Create custom 1200x630 design
3. Use Kawa Note colors (teal + cyan)
4. Add logo and text
5. Download as PNG

**Option C: Command Line (ImageMagick)**
```bash
# Create OG image
convert -size 1200x630 xc:'#0f766e' \
  -fill white -pointsize 60 -font Geist-Sans-Bold \
  -gravity center -annotate +0+100 "Your notes. Your connections. Only yours." \
  og-image.png

# Create square variant
convert -size 800x800 xc:'#0f766e' \
  -composite logo.svg -gravity center -composite \
  og-image-square.png
```

#### 3. Create manifest.json
‚úÖ Already created in Phase 1

#### 4. Create robots.txt
‚úÖ Already created in Phase 1  
**Location:** `frontend/public/robots.txt`

#### 2. Update index.html
**File:** `frontend/index.html`

**Add to `<head>` section:**
```html
<!-- Character encoding and viewport -->
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=5.0" />

<!-- Title and description -->
<title>Kawa Note - Encrypted Note-Taking with Knowledge Graph</title>
<meta name="description" content="End-to-end encrypted note-taking app with semantic connections. Capture ideas, organize in folders, build your knowledge graph. Your notes. Your connections. Only yours." />

<!-- SEO and indexing -->
<meta name="robots" content="index, follow, max-image-preview:large, max-snippet:-1, max-video-preview:-1" />
<meta name="googlebot" content="index, follow" />

<!-- Canonical URL -->
<link rel="canonical" href="https://kawanote.app" />

<!-- Theme color -->
<meta name="theme-color" content="#0f766e" />

<!-- Apple mobile web app -->
<meta name="apple-mobile-web-app-capable" content="yes" />
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent" />

<!-- Open Graph -->
<meta property="og:title" content="Kawa Note - Your Notes. Your Connections. Only Yours." />
<meta property="og:description" content="End-to-end encrypted notes with semantic connections. Build your knowledge graph privately." />
<meta property="og:type" content="website" />
<meta property="og:url" content="https://kawanote.app" />
<meta property="og:image" content="https://kawanote.app/og-image.png" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
<meta property="og:site_name" content="Kawa Note" />

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:site" content="@kawanote" />
<meta name="twitter:title" content="Kawa Note - Encrypted Note-Taking" />
<meta name="twitter:description" content="End-to-end encrypted notes with semantic connections." />
<meta name="twitter:image" content="https://kawanote.app/og-image.png" />

<!-- Favicon -->
<link rel="icon" type="image/svg+xml" href="/icon.svg" />
<link rel="icon" type="image/x-icon" href="/favicon.ico" />
<link rel="apple-touch-icon" href="/icon.svg" />

<!-- Manifest -->
<link rel="manifest" href="/manifest.json" />

<!-- JSON-LD Structured Data -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "SoftwareApplication",
  "name": "Kawa Note",
  "description": "End-to-end encrypted note-taking app with semantic connections and knowledge graph capabilities.",
  "url": "https://kawanote.app",
  "applicationCategory": "ProductivityApplication",
  "image": "https://kawanote.app/og-image.png"
}
</script>
```

### Setup Steps

#### 1. Google Analytics 4
1. Go to https://analytics.google.com/
2. Create new property for `kawanote.app`
3. Get tracking code (G-XXXXXXXXXX)
4. Add to index.html:
   ```html
   <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
   <script>
     window.dataLayer = window.dataLayer || [];
     function gtag(){dataLayer.push(arguments);}
     gtag('js', new Date());
     gtag('config', 'G-XXXXXXXXXX');
   </script>
   ```
5. Set up custom events (signup, login, note_created, etc.)

#### 2. Google Search Console
1. Go to https://search.google.com/search-console
2. Add property: `https://kawanote.app`
3. Verify ownership (DNS, HTML file, or Google Analytics)
4. Submit sitemap: `https://kawanote.app/sitemap.xml`
5. Set up email alerts
